Get the definitions

Accessing DefinitelyTyped is a breeze. Simply download the definitions from GitHub or use one of our supported package managers.

From GitHub

DefinitelyTyped is open source hearted and driven by a great GitHub community. GitHub is the home of all our definitions - just grab the definitions you need, download them to your TypeScript project and start coding. GitHub is also the place to work on definitions by submitting pull requests or reporting issues.

Using Node.js? Say hello to TSD!

The TypeScript Definition Manager or, in short, TSD is a powerful utility that allows you to query the repository and install definitions directly from your command line. TSD keeps track of all used definitions and can update them for you automatically.

> Find out more about TSD

Using VisualStudio? Keep using NuGet!

All DefinitelyTyped definitions are also available through the NuGet package manager right from within your VisualStudio IDE. We push changes from our repositories immediately to NuGet so you don't have to worry about outdated definitions.

Why use DefinitelyTyped?

DefinitelyTyped is the repository for high quality TypeScript type definitions. It contains more than 500 definitions for the most important libraries out there.

Code written in TypeScript includes additional typing information on properties, functions and classes that allow the compiler to perform advanced syntax checks. This ensures that the final application is running stable and smoothly and can be scaled easily.

Other libraries like jQuery or Angular don't cary this meta data as they are written in plain JavaScript. A TypeScript definition file builds a bridge between those two worlds, it contains only the typing information the JavaScript library is missing. You may compare it to a header file in C/C++.

As most of the available libraries don't provide their own definitions, DefinitelyTyped has been introduced to be a central place TypeScript developers use to create and share definition files.

Type checking

With definition files for external libraries the TypeScript compiler is able to check your code against otherwise unknown dependencies and provides useful compile time errors.

Better IDE support

Get more productive by utilizing the full functionality of your IDE like code-completion or inline documentation. Be sure to check our list of editors with advanced TypeScript support.

Open source community

Stop writing custom, partial definitions for your projects. Access and share definitions backed by a great open source community that maintains and updates the repository.
